# FD01 – Definición del Proyecto

## Nombre del Proyecto
Dashboard Electoral Presidencial del Perú – BI

## Integrantes
- Diego Chara Apaza
- Mary Luz Chura Ticona

## Descripción
Desarrollar un dashboard interactivo basado en Inteligencia de Negocios para analizar los resultados de las elecciones presidenciales del Perú y comparar los planes de gobierno de los candidatos.

## Objetivo General
Desarrollar un sistema de visualización de datos electorales mediante herramientas BI.

## Objetivos Específicos
- Recolectar datos electorales confiables
- Procesar y limpiar la información
- Diseñar un modelo de datos
- Crear dashboards interactivos
- Analizar resultados y tendencias

## Herramientas
- Power BI
- GitHub
- Excel / Google Sheets
